# 💸 Planificador de Gastos Mensual

Aplicación web sencilla desarrollada con **HTML, CSS y JavaScript puro**, diseñada para ayudar a gestionar un presupuesto mensual dividiéndolo por semanas y registrando gastos de forma rápida.

## ✨ Características

* Introducción del salario mensual.
* Gestión de gastos fijos ilimitados (alquiler, luz, internet, etc.).
* Número de semanas configurable.
* Cálculo automático del dinero disponible tras descontar los gastos fijos.
* Reparto automático del presupuesto entre las semanas configuradas.
* Registro de gastos semanales con concepto e importe.
* Cálculo en tiempo real del saldo restante por semana.
* Eliminación de gastos fijos y gastos semanales mediante botones con icono de papelera.
* Navegación entre semanas mediante paginación.
* Persistencia de datos mediante **LocalStorage**.
* Diseño responsive compatible con dispositivos móviles y escritorio.

## 🚀 Cómo usar

1. Introduce tu salario mensual.
2. Añade todos tus gastos fijos.
3. Configura el número de semanas que quieres utilizar.
4. Navega entre las semanas utilizando los botones de navegación.
5. Registra los gastos realizados en cada semana.
6. Consulta el saldo disponible actualizado automáticamente.

Todos los datos se guardan automáticamente en el navegador, por lo que no se perderán al cerrar o recargar la página.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API
